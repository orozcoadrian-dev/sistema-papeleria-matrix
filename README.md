# DESCRIPCIÓN

**Papelería Matrix** es un sistema de gestión integral desarrollado para modernizar y optimizar los procesos administrativos, comerciales y operativos del establecimiento. El proyecto busca reemplazar procesos manuales mediante una plataforma tecnológica capaz de centralizar la administración del negocio, facilitando el control de la información y mejorando la eficiencia en las actividades diarias.

La aplicación integra la gestión de inventario, productos, operadores del cibercafé, ventas y autenticación de usuarios dentro de una arquitectura organizada, escalable y mantenible. Su desarrollo sigue buenas prácticas de ingeniería de software, Programación Orientada a Objetos (POO) y el patrón de arquitectura MVC, garantizando un sistema preparado para futuras ampliaciones y nuevas funcionalidades.

---

# OBJETIVOS

El Sistema de Papelería Matrix tiene como finalidad fortalecer la administración del negocio mediante una solución informática que permita optimizar los procesos internos y mejorar el control de la información.

Los objetivos principales del proyecto son:

- Optimizar los procesos administrativos y comerciales.
- Automatizar el control de inventario y existencias.
- Gestionar eficientemente los productos ofrecidos por el establecimiento.
- Centralizar la administración de operadores del cibercafé.
- Registrar y controlar las ventas realizadas diariamente.
- Reducir errores derivados de procesos manuales.
- Implementar una arquitectura limpia, mantenible y escalable para facilitar futuras mejoras.

# FUNCIONALIDADES

### GESTIÓN DE INVENTARIO

Permite controlar las existencias disponibles dentro del establecimiento mediante el registro automático de entradas y salidas de productos, ofreciendo un seguimiento preciso del stock.

- Control de stock.
- Registro de entradas y salidas.
- Alertas de disponibilidad.
- Actualización automática del inventario.

### ADMINISTRACIÓN DE PRODUCTOS

Facilita la organización y administración del catálogo de productos comercializados por la papelería, permitiendo mantener la información siempre actualizada.

- Registro de productos.
- Clasificación por categorías.
- Consulta y edición de información.
- Administración de precios.

### GESTIÓN DE OPERADORES DE CIBERCAFÉ

Administra el personal encargado del funcionamiento del servicio de cibercafé, permitiendo mantener un control organizado sobre los operadores autorizados.

- Registro de operadores.
- Administración de información.
- Control de acceso.

### GESTIÓN COMERCIAL

Permite registrar todas las ventas efectuadas dentro del establecimiento, almacenando un historial organizado de las transacciones realizadas para su posterior consulta.

- Registro de ventas.
- Control de movimientos comerciales.
- Historial de ventas.

### SEGURIDAD

Implementa mecanismos de autenticación y autorización para garantizar que únicamente los usuarios registrados puedan acceder a las funcionalidades correspondientes del sistema.

- Inicio de sesión.
- Gestión de usuarios.
- Control de acceso.
