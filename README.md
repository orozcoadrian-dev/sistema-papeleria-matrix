# 📖 Descripción

**Papelería Matrix** es un sistema de gestión integral desarrollado para modernizar y optimizar los procesos administrativos y comerciales del establecimiento.

La plataforma centraliza la administración de productos, inventario, proveedores, clientes, usuarios y ventas, permitiendo una gestión eficiente mediante una arquitectura robusta, escalable y orientada a buenas prácticas de desarrollo de software.

---

# 🎯 Objetivos

- Optimizar los procesos internos del negocio.
- Automatizar el control de inventario.
- Gestionar ventas y servicios de manera organizada.
- Centralizar la información administrativa.
- Facilitar el crecimiento tecnológico del establecimiento.
- Implementar una arquitectura limpia, mantenible y escalable.

# 🏗 Arquitectura del Proyecto

El sistema está construido siguiendo el patrón de arquitectura **MVC (Model - View - Controller)**, promoviendo una clara separación de responsabilidades y facilitando el mantenimiento del código.

```

Frontend
│
├── HTML5
├── CSS3
└── JavaScript

↓

Controladores (Controllers)

↓

Lógica de Negocio (Models)

↓

Persistencia de Datos (DAO)

↓

MySQL

```
